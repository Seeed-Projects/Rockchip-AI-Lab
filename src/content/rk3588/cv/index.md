# Computer Vision Research

Our Computer Vision research focuses on developing advanced algorithms and applications using Rockchip processors.

## Research Areas

### Image Recognition
We develop state-of-the-art image recognition models optimized for Rockchip hardware, leveraging the powerful processing capabilities of RK3588 and RK1820.

### Object Detection
Our object detection algorithms are designed to run efficiently on edge devices powered by Rockchip processors, enabling real-time applications.

### Scene Understanding
We explore advanced techniques for scene understanding that can operate effectively within the power and performance constraints of embedded systems.

## Hardware Optimization

With the powerful processing capabilities of RK3588 and RK1820, we're pushing the boundaries of what's possible in image recognition, object detection, and scene understanding.

The RK3588 features an octa-core 64-bit processor with integrated NPU for AI applications, making it ideal for complex computer vision tasks.

The RK1820 is a high-performance AI processor optimized for edge computing, perfect for lightweight CV applications.
